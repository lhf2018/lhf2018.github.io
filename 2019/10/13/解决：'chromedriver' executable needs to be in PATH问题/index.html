解决：'chromedriver' executable needs to be in PATH的问题

发表于 2019-10-13 更新于 2019-10-16 分类于 开发工具 阅读次数： Valine：
  • 查看设置-帮助-关于Google Chrome中的版本，如77.0.3865.90
  • 需要下载chrome的webdriver
    找到对应的版本，下载
  • 把webdriver放到chrome的安装文件夹中
    C:\Program Files (x86)\Google\Chrome\Application
  • 把webdriver也放入python的文件夹中
  • 在环境变量的path中添加C:\Program Files (x86)\Google\Chrome\Application的环境变量

成功解决
